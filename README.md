# node_project

您好，這是我所寫的一個課程系統專案

## 專案介紹

**此專案為課程系統的後端專案，專案中有提供許多 API。**

* 資料庫: 使用 MongoDB
* 註冊登入系統 API : 註冊後會發送認證信確認信箱資訊。
* 課程的 CRUD 應用 : 透過會員 JWT token 驗證後，才可執行資料操作。 
* 部屬平台: 以 Render 部屬而成。
* 提供 Swagger 文件
